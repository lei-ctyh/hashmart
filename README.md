## 介绍
HashMart（哈希玛特）是首款开源的面向生产的高性能、易开发的盲盒系统。它包含：首页看板、商品管理、订单管理、盲盒管理、系统管理、会员管理、权限管理、记录管理，并且拥有完善的搭建使用手册和接口文档。是帮助您快速落地盲盒商城系统的不二之选。
